jQuery 对象

DOM对象它有属性如.tagName，它有方法如.appendChild()。

jQuery对象考虑了浏览器兼容性，如target.innerHTML = ...可能在IE中无法正常工作。
DOM对象-->jQuery对象：$(target)。
统一使用$(target).html('...')则解决了这个问题。

又如在某一元素之后插入元素。JSDOM操作target.parentNode.insertBefore(new, target) 使用jQuery： $(target).after(new)

使用jQuery选中/创建的为jQuery对象，var hs = $('h1')将选中所有h1元素。 使用 hs.length可查看元素个数。

var h = $('h1').eq(0)返回首个h1 jQuery元素。
和 var h = $('h1')[0]等效。
var h = $('h1').get(0)则返回一个DOM对象。

jQuery对象并不动态反映文档结构，$('p')并不会在添加p元素后自动更新， 也就是说一旦创建其数据就确定了，除非重新选择。

父、子、兄弟

$( "span.subchild" ).parent();选中其直接父元素。

$( "span.subchild" ).parents( "div.parent" )选中所有符合要求的父元素。

$( "span.subchild" ).parentsUntil( "div.grandparent" );返回所有不符合要求的父元素。

$( "span.subchild" ).closest( "div" )返回最接近的一个父元素。但是注意该元素本身也在搜索范围内。

$( "div.grandparent" ).children( "div" )返回所有符合要求的直接子元素。

$( "div.grandparent" ).find( "div" )返回所有符合要求的后代元素。

兄弟元素选择$( "div" ).next() $( "div" ).prev() $( "div" ).nextAll() $( "div" ).prevAll() $( "div" ).nextAll().last()

数据方法。

使用JavaScript，可将数据绑定到DOM对象上，但可能有内存泄漏问题。 使用jQuery可很容易的存储数据到jQuery对象。

$( "#myDiv" ).data( "keyName", { foo: "bar" } )
$( "#myList li" ).each(function() { var li = $( this ); var div = li.find( "div.content" ); li.data( "contentDiv", div ); })

通用方法

$.trim(' hello there ')删除头&尾空格。
$.each([1,2,3], function(...))迭代对象。
$.inArray(1, [1,2,3])判断数组是否包含元素。
$.extend(fo, so)更新fo，并返回更新后的对象。 类似python词典的update。
$.extend({}, fo, so)将只返回更新后的对象而不改变原对象。
$.proxy( myf, myo )将使myf的this指向myo， 类似bind的作用。

index()方法

$('#name').index()返回id为name的对象的在其父元素中的索引。 $('#name').index('div')返回id为name的div的在所有div元素中的索引。 $('div').index($('#name'))与上同，返回id为name的div的在所有div元素中的索引。
