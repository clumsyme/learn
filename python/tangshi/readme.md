# tangshi
This is a simple chatter bot about Tang poetry.
It should works in py2k.

## Example
```
python robot.py s
```
will launch the app for sentences dialogue.
then
```
asking:君不见
```
will response
```
黄河之水天上来
```

## Else
```
python robot.py a
```
will launch the app for author dialogue.
then
```
asking:李白
```
will response
```
夜思
床前明月光
疑是地上霜
举头望明月
低头思故乡
```
